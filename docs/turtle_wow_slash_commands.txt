SLASH COMMANDS

CHAT[]

/say message - Sends a message to people around you. (/ s)

/whisper name message - Sends a private message. (/ w)

/yell message - Sends a general message. (/ y)

/party message - Sends a message to your group. (/ p)

/raid message - Sends a message to your raid group.

/guild message - Sends a message to your guild.

/reply message - Sends the message to the last message that sent you.
(/ r)

/ignore name - Adds a player to the ignore list.

/unignore name - Removes the player from the ignore list.

/emote message - Equivalent to / me in mIRC. (/ em)

/afk [message] - Switches to "away from keyboard" mode.

/dnd [message] - Switches to "do not disturb" mode.

/join channel - Join / create a chat channel.

/leave channel â€” Disconnects from the chat channel.

GROUP[]

/invite [name] - Invites the selected player to the group, or
specified in the parameter. (/ inv)

/uninvite name - Removes the specified character from your group.

/ffa - Sets the loot type to "free for all".

/master [name] - Sets the type of loot Master Loot, the specified
character is put master looter

/roundrobin - Sets the loot type to "round robin".

/trade - Sends a request to trade the selected character.

GUILD[]

/ginfo - Displays information about your guild.

/g message - Sends a message to members of your guild.

/o message - Sends a message to your guild officers.

/ginvite name - Invites a player to your guild.

/gremove name - Kicks a player out of your guild.

/gpromote name - Increases the rank of the selected player.

/gdemote name - Lowers the rank of the selected player.

/gmotd message - Sets the welcome message.

/gquit - Leave the guild.

/groster - guards roster (officers and leaders only)

/gleader name - Change guild leader.

/gdisband - Disbands your guild (guild leader only)

INTERACTION[]

/cast name (Rank X) - Casts a spell with the specified name, level X.

/assist [name] - Helps attack the chosen target, or character [name].

/inspect - Displays information about the selected character
(inventory, stats).

/target name - Selects the specified character.

/duel [name] - Calls to the duel of the selected character, or the
name of which is indicated in brackets.

/yield - Surrender to a duel

/pvp - Enables the PVP flag for 5 minutes.

MISCELLANEA[]

/who - Displays a list of players around you.

/played - Shows the time spent in the game.

/time - Displays the current game time.

/sit - your character sits down, thereby, accelerating the
regeneration of health.

/stand - Makes your character stand up.

/random [upper limit] - Generates a random number from 1 to the
specified parameter.

/logout - Disconnects from the game.

/exit - Out of the game.

RUN/SCRIPT[]

/run and /script are interchangeable so if you want more characters
(max 255) use /run
